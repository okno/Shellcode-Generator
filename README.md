# Shellcode-Generator

## Language 
Ruby

## Description
ruby > c/c#, Convert .bin into C or C# shellcode

## Dependencies
none :-)

## Howto:
Generate C Shellcode from BIN:
```ruby shellcode_generator.rb file.bin c ```

Generate C# Shellcode from BIN:
``` ruby shellcode_generator.rb file.bin cs ```
